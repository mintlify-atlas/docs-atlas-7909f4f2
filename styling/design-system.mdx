# Design System

The ZOODOM design system is built using CSS custom properties (variables) defined in `global.css`. All colors, typography, and design tokens are centralized for consistency.

## Color Palette

All colors are defined as CSS variables in the `:root` selector, making them globally accessible throughout the application.

The primary brand color used for buttons, links, and key UI elements.


## Naming Convention

All CSS variables follow a consistent naming pattern:

```
--color-{category}-{name}-{shade}
```

### Categories:
- **brand** - Primary brand colors
- **accent** - Secondary brand colors
- **extended** - Additional palette colors
- **neutral** - Grays, blacks, whites

### Shades:
- **900** - Darkest
- **800-100** - Gradual lightening
- **50** - Lightest

<Tip>
  Higher numbers (900) are darker, lower numbers (50) are lighter. This convention makes it intuitive to select shades.
</Tip>

## Typography

### Font Family

The entire site uses the **Onest** font family:

```css
html, body {
    font-family: 'Onest', sans-serif;
    background-color: var(--color-neutral-white-900);
    color: var(--color-neutral-black-900);
}
```

**Location:** `src/assets/global.css:64-68`

Onest is loaded from Google Fonts with variable weights (100-900):

```html
<link href="https://fonts.googleapis.com/css2?family=Onest:wght@100..900&display=swap" rel="stylesheet">
```

## Button Styles

The `.btn-brand` class provides the primary button styling:

```css
.btn-brand {
    background-color: var(--color-brand-green-800);
    color: var(--color-neutral-white-900);
    border: none;
}
.btn-brand:hover {
    background-color: var(--color-brand-green-700);
    color: var(--color-neutral-white-900);
}
.btn-brand:active {
    background-color: var(--color-brand-green-600);
    color: var(--color-neutral-white-900);
}
```

**Location:** `src/assets/global.css:88-100`

### Button States:
- **Default**: `--color-brand-green-800` (#246C32)
- **Hover**: `--color-brand-green-700` (#388E49)
- **Active**: `--color-brand-green-600` (#51B063)

## Global Reset

The design system includes a CSS reset for consistency:

```css

    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

**Location:** `src/assets/global.css:59-63`

## Using Design Tokens

### In CSS

```css
.custom-element {
    background-color: var(--color-brand-green-800);
    color: var(--color-neutral-white-900);
}
```

### In JavaScript/Astro

```javascript
<div style={`background-color: var(--color-brand-green-800)`}>
  Content
</div>
```

<Warning>
  Always use CSS variables instead of hardcoded hex values to ensure design consistency and make future updates easier.
</Warning>
