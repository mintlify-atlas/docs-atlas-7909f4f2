## Overview

The ZOODOM navigation is a responsive Bootstrap 5 navbar with mobile-friendly collapsible menu. It features the ZOODOM logo and five main navigation links.

<Info>
  The navigation is implemented directly in `~/workspace/source/src/pages/index.astro:9-50` using Bootstrap's navbar component.
</Info>

## Navigation Structure

The complete navbar implementation from `~/workspace/source/src/pages/index.astro:9-50`:

```astro
<nav class="navbar navbar-expand-lg navbar-light">
  <section class="container px-3 px-lg-5 py-3">
    <a class="navbar-brand me-0" href="#">
      <!-- Móvil -->
      <img
        src={logo.src}
        class="img-fluid d-inline d-sm-none"
        width="80"
        alt="Logo"
      />
      <!-- Tablet/Desktop -->
      <img
        src={logo.src}
        class="img-fluid d-none d-sm-inline"
        width="110"
        alt="Logo"
      />
    </a>

    <button
      class="navbar-toggler border-0"
      type="button"
      data-bs-toggle="collapse"
      data-bs-target="#mainNavbar"
      aria-controls="mainNavbar"
      aria-expanded="false"
      aria-label="Toggle navigation"
    >
      <span class="navbar-toggler-icon"></span>
    </button>

    <section class="collapse navbar-collapse" id="mainNavbar">
      <ul class="navbar-nav ms-lg-auto mb-2 mb-lg-0 gap-lg-4 pt-3 pt-lg-0">
        <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Inicio</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Especies</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Galeria</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Blog</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Contacto</a></li>
      </ul>
    </section>
  </section>
</nav>
```

## Navigation Links

The navbar contains five main navigation items (`~/workspace/source/src/pages/index.astro:42-46`):

| Link | Description | Status |
|------|-------------|--------|
| **Inicio** | Home page | Active by default |
| **Especies** | Animal species directory | - |
| **Galeria** | Photo gallery | - |
| **Blog** | News and articles | - |

<Note>

## Mobile Toggle Button

The hamburger menu button for mobile navigation (`~/workspace/source/src/pages/index.astro:28-38`):

```astro
<button
  class="navbar-toggler border-0"
  type="button"
  data-bs-toggle="collapse"
  data-bs-target="#mainNavbar"
  aria-controls="mainNavbar"
  aria-expanded="false"
  aria-label="Toggle navigation"
>
  <span class="navbar-toggler-icon"></span>
</button>
```

### Toggle Button Attributes

- **`data-bs-toggle="collapse"`** - Activates Bootstrap's collapse JavaScript plugin
- **`data-bs-target="#mainNavbar"`** - Targets the collapsible navigation section
- **`aria-controls="mainNavbar"`** - Accessibility: indicates what element is controlled
- **`aria-expanded="false"`** - Accessibility: indicates collapsed state
- **`aria-label="Toggle navigation"`** - Accessibility: descriptive label for screen readers
- **`border-0`** - Removes default button border

## Bootstrap Classes Explained

### Navbar Container Classes

```astro
<nav class="navbar navbar-expand-lg navbar-light">
```

- **`navbar`** - Base navbar component class
- **`navbar-expand-lg`** - Navbar collapses below large breakpoint (992px)
- **`navbar-light`** - Light color scheme for navbar text/icons

### Container Spacing

```astro
<section class="container px-3 px-lg-5 py-3">
```

- **`container`** - Responsive fixed-width container
- **`px-3`** - Horizontal padding of 1rem (default)
- **`px-lg-5`** - Horizontal padding of 3rem on large screens
- **`py-3`** - Vertical padding of 1rem

### Navigation List Classes

```astro
<ul class="navbar-nav ms-lg-auto mb-2 mb-lg-0 gap-lg-4 pt-3 pt-lg-0">
```

- **`navbar-nav`** - Full-height navigation list
- **`ms-lg-auto`** - Left margin auto on large screens (pushes menu to right)
- **`mb-2`** - Bottom margin of 0.5rem (default)
- **`mb-lg-0`** - No bottom margin on large screens
- **`gap-lg-4`** - Gap of 1.5rem between items on large screens
- **`pt-3`** - Top padding of 1rem (default)
- **`pt-lg-0`** - No top padding on large screens

## Responsive Behavior

### Desktop (≥ 992px)
- Logo displays at 110px width
- Navigation links appear horizontally in a row
- Links are right-aligned (`ms-lg-auto`)
- 1.5rem gap between navigation items
- Toggle button is hidden

### Mobile (< 992px)
- Logo displays at 80px width
- Toggle button appears (hamburger icon)
- Navigation collapses into a vertical menu
- Menu appears below the navbar when toggle button is clicked

<Warning>
  The navbar uses Bootstrap 5's JavaScript collapse plugin. Ensure Bootstrap JS is loaded (included in Layout.astro).
</Warning>

## Customization Example

To add a new navigation link:

```astro
<ul class="navbar-nav ms-lg-auto mb-2 mb-lg-0 gap-lg-4 pt-3 pt-lg-0">
  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Inicio</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Especies</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Galeria</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Blog</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Contacto</a></li>
  <!-- New link -->
  <li class="nav-item"><a class="nav-link" href="/eventos">Eventos</a></li>
</ul>
```

## Accessibility Features

- **Semantic HTML**: Uses `<nav>` element for navigation landmark
- **ARIA labels**: Toggle button includes `aria-label`, `aria-controls`, and `aria-expanded`
- **Active state**: Current page marked with `aria-current="page"`
- **Alt text**: Logo images include descriptive alt text
- **Keyboard navigation**: All links are keyboard accessible

<Tip>
  The navbar follows Bootstrap 5 accessibility best practices out of the box.
</Tip>
